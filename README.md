## Caso queira instalar o projeto em sua máquina
1- Clone o repositório<br/>
2- Abra o repositório local pelo terminal<br/>
3- Execute o comando <b>yarn install</b><br/>
4- Inicie o servidor local com <b>yarn start</b><br/>
